## How to setup a local relay node
(This is just a basic libp2p-js node)

1. 
Install Node >= 16

2. 
Install CLI
```sh
npm install -g @dao-xyz/peerbit-node
```
3. 
```sh
peerbit start --relay &
```

Ending with '&' to start a background process
