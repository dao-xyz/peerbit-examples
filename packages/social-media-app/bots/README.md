# Social media bots (WIP)

This folder contains a small framework for running “normal user” bots against the Giga social-media network, plus example bots: `JokeBot` and `NewsBot`.

## What’s included

- `@giga-app/bot-kit` (`packages/social-media-app/bots/kit`): shared utilities for building bots (runner, posting helpers, arg parsing).
- `@giga-app/joke-bot` (`packages/social-media-app/bots/joke-bot`): the `JokeBot` program variant.
- `@giga-app/news-bot` (`packages/social-media-app/bots/news-bot`): the `NewsBot` program variant (NewsAPI.ai + OpenAI).
- `@peerbit/joke-bot-cli` (`packages/social-media-app/bots/joke-bot-cli`): CLI docs: `packages/social-media-app/bots/joke-bot-cli/README.md`.
- `@peerbit/news-bot-cli` (`packages/social-media-app/bots/news-bot-cli`): CLI docs: `packages/social-media-app/bots/news-bot-cli/README.md`.
